Clash Royale Deck Matchup Visualizations

This project visualizes **Clash Royale** deck matchups using **PCA** and **t-SNE** projections. Each cluster represents a unique set of deck matchups, allowing you to explore the relationships between different strategies. Use the links below to view the interactive charts, where each data point corresponds to a specific matchup.

  • Winner's First Alphabetical Win Condition
  • Loser's First Alphabetical Win Condition
  • Average Elixir Cost Difference
  • Tuple of Winner and Loser First Alphabetical Win Condition
  • Winner's 100 Closest Cards
  • Loser's 100 Closest Cards
  • Winner's 500 Closest Cards
  • Loser's 500 Closest Cards
  • Winner's 1000 Closest Cards
  • Loser's 1000 Closest Cards

Each chart provides a unique perspective on the relationships between cards, their average elixir cost, and the strategies that lead to victory. Click on a link to explore these insights.
